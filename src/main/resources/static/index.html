/import-csv (POST)

The /import-csv endpoint provides a powerful API to facilitate the seamless import of leads generated from CSV files into your database. To ensure smooth integration, it is essential to adhere to the specific format requirements for the CSV file.

CSV Format Requirements

The CSV file should strictly follow the format outlined below:

ID First Name Last Name Email Phone Gender IP Address
1 John Doe john.doe@email.com +1-123-456-7890 Male 192.168.1.1
2 Jane Smith jane.smith@email.com +1-987-654-3210 Female 10.0.0.2

Monitoring File Process Status

Upon successful upload, the API initiates the processing of your data. During this period, the file process status is your window into the system's progress. It is crucial to check this status to ensure that your file has been imported successfully.

The file process status can indicate whether the import was successful or encountered any issues. Should any discrepancies arise, the system will provide relevant error messages and details to assist you in resolving any potential challenges.

By consistently monitoring the file process status, you can guarantee that your imported data seamlessly integrates into your database, maintaining the integrity and quality of your lead records.
